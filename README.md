# HUD FiveM — Gestion de la faim et de la soif avec UI Vue 3

Ce projet est un HUD complet en cours de développement pour FiveM, conçu pour améliorer l’expérience des joueurs grâce à une interface utilisateur moderne et personnalisable.

## Fonctionnalités principales

- Affichage dynamique des jauges de **faim** et **soif**.  
- Système de traduction multilingue intégré pour une meilleure accessibilité.  
- Sauvegarde des paramètres personnalisés du HUD en base de données, permettant aux joueurs de conserver leurs réglages entre les sessions.  
- Interface utilisateur réalisée avec **Vue 3** pour une expérience fluide et réactive.

## Technologies utilisées

- **LUA** pour la logique serveur et client sur FiveM.  
- **SQL** pour la gestion de la base de données.  
- **Vue 3** pour la partie interface utilisateur.

## Fonctionnalités à venir

- Intégration des jauges de santé et d’armure.  
- Compteur de vitesse.  
- Améliorations de l’interface et nouvelles options de personnalisation.
- Compatible QbCore
